Mis Habilidades

Estas son algunas de las áreas en las que tengo experiencia.

Fronted

Dominio en la creacion de interfaces web modernas usando HTML, CSS y JS. Asi como tecnologias de apoyo como: Bootstrap, fontawesome, sweetAlert, entre otros.

Backend

Experiencia en el desarrollo de la lógica del servidor utilizando lenguajes como Java, Python, C++ y JavaScript. Capacidad para crear APIs, manejar peticiones, estructurar la lógica de negocio y asegurar el correcto funcionamiento de las aplicaciones en el lado del servidor.

Database

Conocimientos básicos en el uso de bases de datos relacionales y consultas SQL para proyectos académicos y personales. Sé realizar modelado simple de datos, organizar tablas y aplicar principios de normalización para asegurar un mejor manejo de la información.

Git - Github

Conocimientos básicos en el uso de sistemas de control de versiones con Git, capacidad para realizar tareas fundamentales como inicializar repositorios, realizar commits y subir cambios a GitHub. Familiaridad con GitHub como plataforma para almacenar proyectos y colaborar de forma sencilla.

Ofimática

Conocimientos en el uso de herramientas ofimáticas como Microsoft Word, Excel y PowerPoint para la creación de documentos, análisis de datos y presentaciones. Redes

Sistema Operativos

Experiencia básica en el manejo de sistemas operativos Windows y Linux, instalación de software y configuración de entornos de desarrollo para proyectos académicos y personales.

Cuento con la capacidad de integrarme en equipos de trabajo de manera colaborativa, aportando ideas y apoyando en la consecución de objetivos comunes. Poseo habilidades de comunicación que me permiten expresar conceptos técnicos y no técnicos de forma clara y adaptada al público.
